# CLAUDE.md
This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Common Commands
- Build: npm run build
- Lint: npm run lint
- Run tests: npm run test
- Run a single test: npm run test -- <test-name>

## Code Architecture
The codebase is structured into the following modules:
- `api`: Handles API requests and responses
- `components`: Contains React components
- `containers`: Contains higher-order components
- `actions`: Handles state management and API interactions
- `reducers`: Handles state management and data processing
- `utils`: Contains utility functions

## Important Files
- `README.md`: Contains information about the project and its usage
- `package.json`: Contains project dependencies and scripts

## Development Notes
- To develop in this codebase, make sure to run `npm install` to install dependencies
- Use `npm run build` to build the project
- Use `npm run lint` to lint the code
- Use `npm run test` to run tests
